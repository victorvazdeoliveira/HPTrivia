# HP Trivia
Olá, leitor(a).

Nesse projeto foi desenvolvido um jogo de quizes para iOS sobre Harry Potter. No jogo, as perguntas são feitar apartir de livros escolhidos pelo jogador, o qual vem com 3 opções de livros gratuitas para jogar e mais 4 a serem compradas.

Assista o vídeo do funcionamento do jogo [clicando aqui](https://youtu.be/-eXvkd6Vfwo?si=GQGf97OoKlDvwAIb).

## Funcionalidades
### Informações
Na tela inicial, existe uma um botão que explica como se joga o jogo.

<img width="471" alt="Tela de Informações" src="https://github.com/user-attachments/assets/f097d4d7-3424-4117-9b0e-49bebc58bd88">

### Configurações
O jogador tem a opção de escolher de quais livros irá receber perguntas. Por padrão existem 3 opções de livros gratuitamente a serem escolhidas e mais 4 que precisam ser compradas para serem disponibilizadas.

<img width="471" alt="Tela de Configurações" src="https://github.com/user-attachments/assets/5bd9563d-f554-4b0b-ad12-7646ea99aac2">

<img width="471" alt="Tela de Compra" src="https://github.com/user-attachments/assets/772ed2fa-0f1e-444b-a1d7-0a035245284a">

### Perguntas
Como proposto, existe uma gameplay com uma pergunta e quatro possíveis respostam no qual o jogador deve escolher a correta. Escolhendo a alternativa errada acarretará em uma redução de pontuação recebida sobre a pergunta.

<img width="471" alt="Tela de Quiz" src="https://github.com/user-attachments/assets/ce7a16ca-c48b-45b8-8b21-c12ae3d5dfac">

<img width="471" alt="Tela de Acerto" src="https://github.com/user-attachments/assets/a39cf08d-ebd2-4ad9-afb3-e81c7a773545">


### Dicas
Durante o gameplay, o jogador possui 2 dicas, uma sobre qual alternativa pode ser a certa e outra sobre qual livro é baseado a pergunta.
